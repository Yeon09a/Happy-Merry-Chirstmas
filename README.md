## 게임 소개
‘해피 메리 크리스마스’는 공간플랫폼 기반 체험형 XR 게임입니다.<br/>
프로젝트 전시 시기 12월에 맞추어 크리스마스라는 계절적 테마를 차용해 어린이들의 상상력 자극, 추론을 통한 사고력 향상, 협업에 대한 학습과 성취감 제공을 목적으로 하고 있습니다.<br/>
